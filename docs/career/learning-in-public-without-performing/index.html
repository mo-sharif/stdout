Sustainable growth

Learning in public without performing

Sharing what you learn can be generous. It gets weird when the performance replaces the learning.

~5 min•Career
scroll↓
01 · the shape

The internet rewards certainty

Real learning is awkward: false starts, half models, corrections, and notes that only make sense later. Public platforms reward clean takes. Those incentives can pull you away from curiosity.

notes · learning loop

The useful signal is usually already there.

02 · the move

Publish the useful artifact

Share a debugging note, diagram, checklist, or mistake with enough context that someone else can use it. You do not need to pretend it is the final word.

learning-note.md

A caveat section is a power move.

03 · the check

Keep a private workbench

Not every note needs an audience. Private scratchpads protect the messy part of learning so public sharing can stay honest.

0useful artifact per post
0sections that keep learning honest

04 · the takeaway

Share to clarify, not to cosplay mastery

The best public learning still feels like a person thinking carefully, not a brand defending a throne.
